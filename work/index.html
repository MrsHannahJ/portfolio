Responsive WordPress Site with Customized Theme

This client had an existing site built with the builder provided by 000webhosting. They wanted to retain the free hosting service, which created a few challenges. The free hosting has limited bandwidth and security, so it has sluggish performance and cannot support e-commerce due to security concerns. But, we were able to create some work-arounds and were still able to attain the "customized, dark, art deco" feel that they desired.

I designed and custom built a site that better supported product sales and user experience. I also setup a customized WordPress admin so that the not-so-tech-friendly user would be able to maintain the site into the future without needing to hire a developer.
